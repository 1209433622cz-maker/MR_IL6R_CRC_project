## What is included

This repository package contains a GitHub-ready, public-safe subset of the MR_IL6R_CRC project:

- Revised manuscript with GEO external validation
- Final main figures and supplementary figure
- Supplementary tables
- Figure legends
- GEO independent validation scripts
- Source traceability registries and manifests
- Reference manager files and reference PDF availability manifests
- Workflow records and reports

## What is intentionally not included

Full copyrighted reference PDFs are **not** included in this public GitHub-ready package.  
Use `references/manifests/` and `references/download_scripts/` to document or retrieve open-access references legally.

Large local archives, raw GEO downloads, and private source folders should not be pushed directly to GitHub unless you intentionally use Git LFS or a release asset workflow.
