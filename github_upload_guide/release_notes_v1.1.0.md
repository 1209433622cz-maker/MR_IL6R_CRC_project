## MR_IL6R_CRC manuscript data package v1.1.0

This release updates the public manuscript-supporting repository for the MR_IL6R_CRC project after v51 manual GEO validation and Figure 6 revision.

### Added in v1.1.0

- v51 revised manuscript files with manual GEO validation integration.
- GSE39582 tumour-normal validation results for IL6R expression.
- GEO stage-stratified and survival validation summary tables.
- Supplementary Figure S2 composite GEO manual validation figure.
- Figure 6 v51 candidate generated from traceable single-cell source data.
- Supplementary Table S8 figure source-data/code crosswalk.
- v51 workflow records, rerun scripts, source-data manifests and revision report.

### Main validation result added

In GSE39582, IL6R expression was higher in non-tumoral colorectal mucosa than in tumour samples:

- tumour n = 566
- non-tumoral mucosa n = 19
- median IL6R tumour = 4.064
- median IL6R non-tumoral mucosa = 6.677
- Wilcoxon P = 3.94e-13

Stage-stratified and survival validation analyses did not support IL6R as a robust standalone stage-associated or prognostic marker, reinforcing the manuscript interpretation that IL6R is linked to immune / myeloid / inflammatory microenvironmental context rather than a simple tumour-intrinsic oncogenic marker.
