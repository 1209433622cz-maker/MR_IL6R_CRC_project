## MR_IL6R_CRC manuscript data package v1.0.0

This release contains the revised manuscript-supporting package for the MR_IL6R_CRC project, including:

- Revised manuscript with GEO external validation
- Final main figures
- Supplementary Figure S1
- Supplementary Tables S1-S7
- GEO independent validation scripts
- Figure source-data and traceability registries
- Reference manager files and reference availability manifests

Notes:

- Full reference PDFs are not redistributed in this public repository.
- Public raw datasets should be obtained from their original repositories.
- The GEO validation workflow is documented in the scripts and supplementary tables.
