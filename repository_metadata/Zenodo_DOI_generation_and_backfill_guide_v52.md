# v52 Zenodo DOI workflow for MR_IL6R_CRC v1.1.0

## Goal

Create a citable Zenodo DOI for the GitHub repository release:

`https://github.com/1209433622cz-maker/MR_IL6R_CRC_project/releases/tag/v1.1.0`

## Recommended workflow

1. Make sure the GitHub repository contains the v51/v52 files and has a published release named `v1.1.0`.
2. Log in to Zenodo with GitHub.
3. Open Zenodo GitHub integration settings.
4. Enable the repository `1209433622cz-maker/MR_IL6R_CRC_project`.
5. Create or re-create the GitHub release `v1.1.0` after enabling Zenodo integration if Zenodo does not detect the existing release.
6. Wait for Zenodo to archive the release.
7. Copy the version DOI and record URL.
8. Run `run_v52_03_backfill_manuscript_after_zenodo_doi.ps1` with the DOI and Zenodo record URL.
