# Reference PDFs

This GitHub-ready package intentionally excludes full reference PDFs by default.

Reasons:

1. Some PDFs may be copyrighted or publisher-restricted.
2. GitHub is not ideal for storing large PDF libraries.
3. Public repositories should include reference metadata, DOI manifests, and legal download scripts instead.

Included here:

- Reference manager files (`.bib`, `.ris`)
- Reference audit CSV files
- PDF availability manifests
- Open-access download helper script where available

Keep the complete local reference PDF archive outside the public repository unless you have confirmed redistribution rights.
